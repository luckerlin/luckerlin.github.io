🏠 关于本站

在 2021-01-29 20:45:01，我在“热铁盒网页托管”上（ https://host.retiehe.com/ ）提交了我的第一份博客。它极其简陋，只有简单的文字和背景图片，颇有上世纪的网页风格。但它也花费了我大量时间。因需会员才能上传图片，所以就放弃花心思维护了。

2022年的暑假，我在冲浪时了解到 GitHub 上能建网站，又燃起了我对创建博客站的兴趣。免费的 GitHub Pages 成功吸引了我，我把之前的博客进行了脱胎换骨般的修改。并在 Gridea 静态博客写作客户端 上传博客至 GitHub Pages，使用 Gridea Theme 中的 Notes 模板。

但因软件功能太少，且已不怎么更新，所以换用了 WordPress。又因种种原因，换用 Hexo 静态博客架构，使用 Butterfly 模板，进行了大量魔改。魔改来魔改去，又觉得不太喜欢，换成 Volantis。Volantis 功能太多，维护起来麻烦，又改成了现在的 Icarus。（简洁即美，=￣(工)￣= ，魔改的尽头即是默认）

本博客会时不时地更新，内容有一些是有关编程的文章，还有一些对生活的思考想法。目前能想到的就这么多，有时间会做一些有意思的内容，敬请期待！

👨‍💻 博主是谁

博主是一位卑微的学牲，精力和能力有限，但我正尽力地维护博客。
